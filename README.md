# 영수증 N빵 계산기

영수증 사진을 업로드하면 무료 OCR.space API로 영수증 총액을 추출하고,
입력한 인원 수에 따라 더치페이(1/N 분할) 금액을 계산해 주는 웹 애플리케이션입니다.
Node.js 내장 HTTP 서버가 정적 파일을 제공하며, 모든 처리는 브라우저에서만 수행됩니다.

## 주요 기능
- 영수증 이미지 업로드 및 미리보기
- OCR.space(무료) 기반 OCR로 한글 영수증 인식 향상
- 영수증을 여러 장 추가하면 자동 합산 (직접 금액 입력도 가능)
- 합계/총액 키워드를 기반으로 자동 총 결제금액 추출 + 부가세 보정
- 인원 수를 입력하면 1인당 부담 금액 계산 및 몫/잔액 분배 안내
- 총액이 잘못 인식된 경우 직접 수정 가능
- 결과를 클릭 한 번으로 클립보드에 복사

## 기술 스택
- **프론트엔드**: HTML, CSS, Vanilla JavaScript, OCR.space API
- **백엔드**: Node.js (내장 HTTP 모듈로 정적 파일 서빙)

## 개발/실행 방법
1. (선택) 의존성 설치 — 외부 패키지가 없으므로 생략해도 됩니다.
   ```bash
   npm install
   ```
2. 개발 서버 실행
   ```bash
   npm start
   ```
3. 브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

## OCR API 키 설정
- 기본적으로 OCR.space에서 제공하는 데모 키(`helloworld`)를 사용하며, 분당/일일 호출 제한이 매우 낮습니다.
- 개인 무료 계정을 발급받은 뒤 **API 키를 두 번째 섹션의 입력란에 넣으면** localStorage에 저장되어 재사용됩니다.
- API 키를 입력하지 않아도 동작하지만 대량 테스트 시 제한에 걸릴 수 있습니다.

## 사용 팁
- 여러 장의 영수증을 업로드하면 합계가 자동 갱신되며, 각 항목의 금액은 개별적으로 수정할 수 있습니다.
- 영수증에 `공급가액` / `부가세`만 있을 경우 자동으로 합계를 계산하므로 마지막 합계가 없더라도 정확한 총액을 얻을 수 있습니다.
- OCR 결과가 부정확하다면 해당 영수증 항목의 금액 입력란에서 직접 수정하세요.
